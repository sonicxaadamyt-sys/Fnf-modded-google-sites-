# FNF Modded for Google Sites

This is a copy of Friday Night Funkin' (HTML version) prepared for use with Google Sites and custom modifications.

## About
Friday Night Funkin' is a rhythm game created by The Funkin' Crew. This repository contains an HTML/web version.

## Getting Started
- Open `index.html` in a web browser
- Or run `webserv.bat` on Windows for a local server
